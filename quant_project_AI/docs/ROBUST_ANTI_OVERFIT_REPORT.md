## 11. Methodology

### Purged Walk-Forward Windows (Anchored, Train/Val/Test)

```
Window 1: train [0, 35%), [embargo 5 bars], val [35%+emb, 45%), test [45%, 55%)
Window 2: train [0, 45%), [embargo 5 bars], val [45%+emb, 55%), test [55%, 65%)
Window 3: train [0, 55%), [embargo 5 bars], val [55%+emb, 65%), test [65%, 75%)
Window 4: train [0, 65%), [embargo 5 bars], val [65%+emb, 75%), test [75%, 85%)
Window 5: train [0, 75%), [embargo 5 bars], val [75%+emb, 85%), test [85%, 95%)
Window 6: train [0, 80%), [embargo 5 bars], val [80%+emb, 90%), test [90%, 100%)
```

**Embargo**: 5-bar gap between train and val prevents information leakage

Val and Test have **equal width** (10% each). This isolates overfitting from regime change.

### Monte Carlo Price Perturbation

```
N paths: 30, Noise: Gaussian sigma = 0.2% per bar
Perturbed: close (signal), open (fill), high/low (channels)
OHLC validity maintained: low <= min(O,C), high >= max(O,C)
```

### OHLC Shuffle Perturbation

```
N paths: 20
Method: randomly permute {O,H,L,C} per bar, then re-assign:
  max -> H, min -> L, remaining two -> O and C
```

Tests structural dependence: a strategy that relies on 'close is the last
traded price' should degrade when close becomes a random intra-bar price.

### Block Bootstrap Resampling

```
N paths: 20, Block size: 20 bars
Method: resample contiguous blocks with replacement, scale to maintain price continuity
```

Preserves local autocorrelation (momentum, mean-reversion within blocks)
while destroying long-range dependencies. Strategies that only work on a
specific sequence of events will fail.

### Deflated Sharpe Ratio (Bailey & Lopez de Prado, 2014)

```
E[max(SR)] = (1-γ)√(2·ln(N)) + γ/√(2·ln(N))  where N = param grid size
DSR = Φ((SR_obs - E[max(SR)]) / SE(SR))
```

Corrects for the fact that scanning many parameter combinations inflates
the best observed Sharpe. A DSR near 0 means the strategy's performance
is no better than the best of N random noise trials.

### Composite Ranking (8-dimensional)

```
composite = rank(WFE↓) + rank(GenGap↑) + rank(Stability↓) + rank(MC↓)
          + rank(Shuffle↓) + rank(Bootstrap↓) + rank(DSR↓) + rank(Cross↓)
```

### Verdicts

- **ROBUST**: ≥7/8 criteria (WFE>0, Gap<5%, Stab>0.5, MC>0.5, Shuffle>0.3, Boot>0.3, DSR>0.3, Cross>0)
- **STRONG**: 5-6/8 criteria
- **MODERATE**: 3-4/8 criteria
- **WEAK**: <3/8 criteria
