## 2. 新增三层技术详解

### 2.1 Layer 7: OHLC Shuffle (价格角色互换)

**核心思想**: 每根K线的 {Open, High, Low, Close} 四个价格被随机重新分配角色, 然后强制保持有效性 (max→H, min→L, 其余→O和C)。

**统计学原理**:
- 策略依赖 "Close是最后成交价" 这个假设来计算指标 (MA, RSI等)
- 如果将Close替换为当日的任意一个价格 (可能是Open或High), 真正有效的策略不应该完全崩溃
- 这等价于测试: **策略是否过度依赖OHLC的特定标签, 还是捕获了真实的价格运动规律?**

**关键发现**: 大多数策略在OHLC Shuffle下仍能保持盈利 (MA 0.753, MACD 0.770, MomBreak 0.683), 说明它们的逻辑确实基于价格运动趋势而非标签偶然性。

### 2.2 Layer 8: Block Bootstrap (区块自举重采样)

**核心思想**: 将价格序列切成20根K线的区块, 然后有放回地随机重组, 同时维持价格连续性 (通过缩放因子)。

**统计学原理**:
- 保留了区块内部的短期自相关结构 (动量、均值回归)
- 但破坏了长期序列依赖关系
- 如果策略只在特定的事件序列下有效 (如 "2021年牛市→2022年熊市→2023年反弹"), 那它在bootstrap路径上会失败

**关键发现**: **Bootstrap是最严格的过滤器** — 几乎所有策略的Bootstrap分数都接近0。只有MA (0.032), MACD (0.032), MultiFactor (0.047) 略有正值, 说明在完全打乱时间顺序后, 绝大多数策略失去了盈利能力。这揭示了一个重要事实: **技术分析策略的有效性高度依赖于市场的时序结构**。

### 2.3 Layer 9: Deflated Sharpe Ratio (膨胀夏普比校正)

**核心思想**: 当你在N个参数组合中选择最优参数时, 即使所有参数都产生随机收益, "最优"参数也会显示出看似很高的夏普比。DSR校正了这种多重检验偏差。

**计算方式**:
```
E[max(SR)] = (1-γ)·√(2·ln(N)) + γ/√(2·ln(N))   (N = 参数组合数)
DSR = Φ((SR_obs - E[max(SR)]) / SE(SR))
```

**关键发现**: 所有21个策略的DSR几乎都是0, 只有MultiFactor (0.125) 略高。这意味着: 在考虑了我们扫描的271,619个参数组合后, 没有任何策略的样本外夏普比显著超过随机噪声的预期最大值。这是对过拟合最严厉的判决。

## 5. 策略深度分析

### 5.1 MomBreak (排名 #1, STRONG)

**8维表现**:
- WFE +16.8% (6窗口平均, 比V2的4窗口+10.7%提升)

- Bootstrap 0.010 — 勉强为正, 是极少数在bootstrap下不为0的策略
- DSR 0.000 — 参数搜索空间过大导致夏普被完全通胀

**核心优势**: 在所有新层测试中一致表现最均衡。没有任何维度严重失败 (除DSR)。
**使用建议**: **最推荐的单策略选择**。参数建议: lookback 20-60, 突破阈值 0.01-0.08。

### 5.2 MACD (排名 #2, STRONG)

**8维表现**:
- WFE +21.9% — 全场最高! 6窗口验证了更强的盈利能力
- **Shuffle 0.770** — 全场最高的OHLC Shuffle分数
- **Bootstrap 0.032** — 并列最高 (与MA)
- MC 0.908 — 全场最高

**核心优势**: 在每一个维度上都接近最好, 但综合分略逊MomBreak (因为GenGap更大)。
**OHLC Shuffle全场第一**: 这意味着即使把Close换成High或Open, MACD信号依然有效 — 验证了指数平滑移动平均的鲁棒性。

### 5.3 MA (排名 #3, MODERATE → 跃升)

**惊人发现**: 最简单的MA交叉策略在V3中从#6跃升到#3, 原因:
- Shuffle 0.753 — 第二高, 简单移动平均对价格标签不敏感
- Bootstrap 0.032 — 并列最高, 简单逻辑在打乱时序后仍能部分工作
- MC 0.921 — 全场最高!

**教训**: "简单即鲁棒"。MA交叉虽然GenGap较高 (32.3%) 且参数稳定性差 (0.307), 但在最极端的数据扰动测试中表现最好, 说明其底层逻辑 (价格穿越均线 = 趋势转折) 是市场中最基本的规律之一。

### 5.4 MultiFactor (排名 #4, 值得关注)

**为什么值得关注**:
- DSR 0.125 — **全场最高!** 这是唯一一个在Deflated Sharpe下有非零信号的策略
- Bootstrap 0.047 — 全场最高
- GenGap 16.1% — 相对较低, 泛化能力好

**含义**: MultiFactor结合了RSI、动量和波动率三个因子, 多因子模型天然分散了单一因子的噪声, 使其在统计检验中更不容易被认为是偶然。

## 6. Bootstrap: 最严格的过滤器

Block Bootstrap 是本次新增中**最具颠覆性**的测试。结果:

| Bootstrap > 0 | Bootstrap = 0 |
|:---|:---|
| MultiFactor (0.047) | Consensus (0.000) |
| MA (0.032) | MomBrkPlus (0.000) |
| MACD (0.032) | RAMOM (0.000) |
| RSI (0.025) | Keltner (0.000) |
| MomBreak (0.010) | Turtle (0.000) |
| RegimeEMA (0.002) | 等16个策略... |

**解读**: Bootstrap打乱了K线的时间顺序 (虽然每个20-bar区块内的顺序保留), 这意味着:
1. 策略无法依赖 "2021年牛市→2022年熊市" 这种宏观趋势
2. 只有在**任何时间段的微观价格运动中都有效**的策略才能通过
3. 绝大多数策略 (16/21) 完全依赖于特定的市场时序, 一旦打乱就归零

**这解释了为什么没有策略达到ROBUST**: ROBUST要求Bootstrap>0.3, 但最高的只有0.047。

## 7. Deflated Sharpe: 多重检验的惩罚

DSR几乎将所有策略判为0, 原因:
- 每个策略扫描了 ~271,619 个参数组合
- E[max(SR)] 约为 √(2·ln(271619)) ≈ 5.0 (标准化夏普)
- 而样本外的实际夏普远低于此阈值

**实际意义**: 即使MomBreak和MACD在6个窗口上的WFE都很高, 但考虑到我们从27万个参数中选出了"最佳", 这种"最佳"的表现在统计上不显著。

**解决方向**: 减少参数搜索空间 (使用更粗的网格或贝叶斯优化) 可以降低E[max(SR)]惩罚, 从而提高DSR。

## 8. OHLC Shuffle: 验证策略逻辑的本质

Shuffle测试回答了一个根本性问题: **策略是在交易"价格运动"还是在交易"标签"?**

| Shuffle > 0.5 | Shuffle 0.3-0.5 | Shuffle < 0.3 |
|:---|:---|:---|
| MACD (0.770) | RSI (0.445) | KAMA (0.288) |
| MA (0.753) | MultiFactor (0.403) | Turtle (0.318) |
| MomBreak (0.683) | TFiltRSI (0.400) | MESA (0.313) |
| RegimeEMA (0.671) | | Bollinger (0.212) |
| MomBrkPlus (0.627) | | Connors (0.202) |
| RAMOM (0.608) | | ZScore (0.196) |
| DualMom (0.559) | | Donchian (0.149) |
| Consensus (0.541) | | VolRegime (0.070) |
| Keltner (0.534) | | Drift (0.000) |

**规律**: 基于**趋势/动量**的策略 (MA, MACD, MomBreak) 在Shuffle下表现最好, 因为无论Close是"真正的收盘价"还是当日的某个随机价格, 趋势的方向不会改变。

而基于**精确价格水平**的策略 (ZScore, Bollinger, Donchian) 在Shuffle下表现最差, 因为它们依赖于Close精确落在布林带/通道的哪个位置。

## 10. 核心结论

1. **10层系统比7层更严格**: 8个STRONG策略取代了之前的全部MODERATE, 但仍然0个ROBUST。Bootstrap和DSR是最严格的过滤器。

2. **MomBreak和MACD是确认的冠军**: 在从5维扩展到8维、从4窗口扩展到6窗口后, 这两个策略仍然稳居前2。排名变化极小, 说明它们的优势是真实的。

3. **OHLC Shuffle验证了趋势策略的本质**: 趋势跟随策略 (MA, MACD, MomBreak) 不依赖Close的精确定义, 在价格标签互换后仍能工作。这是策略逻辑合理性的直接证据。

4. **Block Bootstrap揭示了所有策略的阿基里斯之踵**: 没有策略能在完全打乱时间顺序后保持盈利 (最高仅0.047)。这意味着技术分析的有效性**根本上依赖于市场的时序结构** — 这既是机会 (市场确实有时序模式), 也是风险 (时序模式可能改变)。

5. **Deflated Sharpe对参数优化的惩罚是毁灭性的**: 扫描27万个参数组合后, 即使最好的策略也不能在统计上证明其夏普比优于随机。解决方案: 减少参数搜索空间或使用更多样本外数据。

6. **"简单即鲁棒"原则得到验证**: MA交叉从#6升至#3, 因为它在最极端的数据扰动下表现最好。过于复杂的策略 (Consensus从#3降至#10) 反而更脆弱。
