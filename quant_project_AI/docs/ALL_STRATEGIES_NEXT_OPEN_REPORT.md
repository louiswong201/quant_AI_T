## 方法与严谨性说明

1. 市价成交采用 next-open，避免同 bar close 成交带来的前视偏差。

2. 参数空间按 [x/2, x, 2x] 构造，并加入必要物理约束（如 fast < slow, RSI 阈值在 0~100）。

3. 选参仅使用训练集，测试集完全独立；报告中单列泛化保持率用于识别过拟合。

4. 成本模型采用偏保守设定，避免回测结果过于乐观。
